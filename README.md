# greendent-tailwindcss
🌿 Greenden - Plant Selling Website

A simple and responsive plant-selling website built using HTML and Tailwind CSS. This project showcases a modern plant store interface with multiple pages and a clean user experience.

🚀 Features
Responsive design using Tailwind CSS
Clean and modern user interface
Home page with featured plant collections
Product page displaying available plants
Contact page for customer inquiries
Mobile-friendly layout
Easy navigation between pages

HTML5
Tailwind CSS
